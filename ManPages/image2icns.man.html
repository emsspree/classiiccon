IMAGE2ICNS(1)             BSD General Commands Manual            IMAGE2ICNS(1)

NAME

     image2icns — Create Mac OS X icns file from an image

SYNOPSIS

     image2icns image_file out_icon

DESCRIPTION

     image2icns creates an Apple icns file from a source image

EXAMPLES

     image2icns image.png out.icns

SEE ALSO

     geticon(1), iconutil(1), image2icns(1), seticon(1)
