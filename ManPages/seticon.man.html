NAME

     seticon — Set the custom icon of Mac file

SYNOPSIS

     seticon iconFile [files ...]

DESCRIPTION

     seticon sets the Mac custom icon of one or more files

EXAMPLES

     seticon icon.icns file1.txt file2.txt

SEE ALSO

     geticon(1), icns2image(1), iconutil(1), image2icns(1)
